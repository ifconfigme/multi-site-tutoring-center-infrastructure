# Multi-Site Tutoring Center Infrastructure

Secure, scalable IT infrastructure design for a simulated non-profit tutoring organization operating three centers in Romeoville, Chicago, and Aurora, Illinois.

## Project Overview

This capstone project focused on designing, documenting, and presenting a multi-site network infrastructure that supports staff, tutors, students, secure remote access, shared resources, and operational continuity.

## Key Features

- Three-site network design for Romeoville, Chicago, and Aurora
- VLAN segmentation for Staff, Tutors, Guests, Servers, and DMZ resources
- IPsec VPN connectivity between locations
- Fortinet firewall/security architecture
- IDS/IPS, DLP, and web filtering recommendations
- Microsoft Sentinel SIEM planning and alerting documentation
- IP addressing and subnetting plan
- Equipment selection and budget justification
- User training and ongoing IT support documentation
- Foundational IT policies including AUP, ACP, IRP, DRP, and change management
