# Security Architecture

This folder summarizes the security design components used in the Multi-Site Tutoring Center Infrastructure Capstone.

## Key Security Areas

- Fortinet firewall architecture
- VLAN segmentation
- IPsec site-to-site VPN connectivity
- IDS/IPS planning
- Data Loss Prevention planning
- Web filtering recommendations
- Microsoft Sentinel SIEM planning
- Access control lists
- Captive portal authentication
- Guest network isolation

## Security Goals

The security design was created to protect internal systems, isolate guest traffic, secure communication between sites, monitor suspicious activity, and support scalable IT operations across Romeoville, Chicago, and Aurora.
