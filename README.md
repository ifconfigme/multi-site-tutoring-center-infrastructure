# multi-site-tutoring-center-infrastructure
Secure multi-site IT infrastructure design for tutoring centers, including VLAN segmentation, IPsec VPNs, Fortinet firewalls, Microsoft Sentinel SIEM, subnetting, access controls, equipment planning, and IT support documentation.
