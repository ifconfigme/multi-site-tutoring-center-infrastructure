## Project Context

This documentation supports a secure multi-site IT infrastructure design for a simulated non-profit tutoring organization operating three locations in Romeoville, Chicago, and Aurora, Illinois.

The project includes:

- Multi-site network design
- VLAN segmentation
- IP addressing and subnet planning
- IPsec VPN connectivity
- Firewall and access control planning
- Microsoft Sentinel SIEM planning
- IDS/IPS, DLP, and web filtering recommendations
- Equipment selection and budget justification
- User training and ongoing IT support documentation
- Foundational IT policy development
